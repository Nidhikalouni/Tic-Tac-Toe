# Tic-Tac-Toe

## Introduction
Tic-Tac-Toe is a simple, classic game for two players (X and O) who take turns marking the spaces in a 3Ã—3 grid. The player who succeeds in placing three of their marks in a horizontal, vertical, or diagonal row wins the game.

This project is a basic implementation of the game using HTML, CSS, and JavaScript.

## Features
- Two-player mode
- Real-time game status updates
- Simple and intuitive UI

## Technologies Used
- HTML
- CSS
- JavaScript
